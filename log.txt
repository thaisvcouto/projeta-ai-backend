1. Apresente aos alunos uma história onde um robô precisa explorar um labirinto para encontrar um tesouro.
2. Explique o conceito de condicionais, mostrando como o robô pode tomar decisões com base em certas condições.
3. Peça aos alunos para desenharem o labirinto e os obstáculos no Scratch.
4. Ajude os alunos a programarem o robô (sprite) para se mover pelo labirinto. Utilize blocos condicionais if-else para definir o comportamento do robô ao encontrar obstáculos.
5. Desafie os alunos a programarem o robô para chegar ao tesouro evitando os obstáculos.
6. Incentive os alunos a testarem e ajustarem o código conforme necessário.

**Código Base:**
```blocks
quando a bandeira verde for clicada
ir para x: (-160) y: (-80)
repita até <tocando [tesouro v]?>
  se tocar na cor [azul] então
    virar para a direção [90 v] graus
  senão
    se tocar na cor [verde] então
      virar para a direção [-90 v] graus
    senão
      seguir em frente por (10) passos
```

Este código simples simula o movimento do robô pelo labirinto, girando à direita se encontrar a cor azul (representando um obstáculo) e à esquerda se encontrar a cor verde, até chegar ao tesouro. Os alunos podem personalizar e aprimorar o código conforme exploram mais a lógica de programação.
